# Minimum Viable Product

## Base

- A web app that will allow you to create notes in an hmtl format without writing html yourself
- The option to copy prebuilt notes that are already in html format

## Specifics

- Starts on the **Note Builder** page, with the option to go to the **Note Snippets** page on the _Nav Bar_

### Note Builder

- **Promt Panel**
  - Starts with a single prompt that allows you to choose the type of appointment (1st, 2nd, 3rd or Post).
  - **Saves user's initials to local storage** and adds them on the next use automatically
- **Generate Code**
  - Displays code as each prompt is selected
  - Can be copied by clicking on it

### Note Snippets

- A list of pre-built notes that can be copied
