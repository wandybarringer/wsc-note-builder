# Prompt Layout

1. **Dropdown** for the appointment option
   - 1st appointment
   - 2nd appointment
   - 3rd appointment
   - Post appointment

## IF 1st Appointment

2. Is it a continuation appointment **checkbox**

3. Intro to Warhead completed **checkbox**

4. Homework completed **checkbox**

5. % of training videos **number input**

6. Worked on **checkboxes** (Options specific to appointment)

7. Assigned homework **checkboxes** (Options specific to appointment)

8. Additional notes **text input**

9. Client started registering business **checkbox**

10. Set next appointment date **text input** or **calendar**

11. Next topic **text input**

12. Signature

## IF 2nd Appointment

2. Is it a continuation appointment **checkbox**

3. Homework completed **checkbox**

4. % of training videos **number input**

5. Worked on **checkboxes** (Options specific to appointment)

6. Assigned homework **checkboxes** (Options specific to appointment)

7. Additional notes **text input**

8. Set next appointment date **text input** or **calendar**

9. Next topic **text input**

10. Signature

## IF 3rd Appointment

2. Is it a continuation appointment **checkbox**

3. Homework completed **checkbox**

4. % of training videos **number input**

5. Worked on **checkboxes** (Options specific to appointment)

6. Assigned homework **checkboxes** (Options specific to appointment)

7. Additional notes **text input**

8. Sent completion form **checkbox**

9. signed or not signed **dropdown**

10. Why not signed? **text input** (Only IF not signed selected on prompt 9)

11. Set next appointment date **text input** or **calendar**

12. Set SM appointment name and date **text input** or **calendar**

13. Sent SM how to guide and enrolled in SM training **checkboxes**

14. Next topic **text input**

15. Signature

## IF Post Appointment

2. Is it a continuation appointment **checkbox**

3. Homework completed **checkbox**

4. % of training videos **number input**

5. Worked on **checkboxes** (Options specific to appointment)

6. Post training checklist **checkboxes**

7. Additional notes **text input**

8. Client live? **dropdown**

9. Additional training requested **dropdown** (Only if live selected on prompt 8)

10. Set continuation Post appt Date **text input** or **calendar** (Only IF not live or additional training requested on prompt 8 or 9)

11. Reminded about SM appointment name and date **text input** or **calendar**

12. Next topic **text input** (Only IF not live or additional training requested on prompt 8 or 9)

13. Signature
