# Final Product

## Base

- A web app that will allow you to create notes in an hmtl format without writing html yourself
- The option to copy prebuilt notes that are already in html format
  - Allowing you to favorite and search snippets

## Specifics

- Starts on the **Note Builder** page, with the option to go to the **Note Snippets** page on the _Nav Bar_

### Note Builder

- **Promt Panel**
  - Starts with a single prompt that allows you to choose the type of **any note** (appt, reshcedule, missed appt, etc.).
  - Has the option to show all prompts at once by clicking a check box on or off
  - **Saves user's initials to local storage** and adds them on the next use automatically
- **Generate Code**
  - Displays code as each prompt is selected
  - Can be **edited**
  - Can be copied by clicking on it

### Note Snippets

- A list of pre-built notes that can be edited **and copied**
- **Search functionality**
- **_Local Storage Required Functionality_**
  - Most recently used note snippets sorted at the top
  - Option to favorite snippets and sort by favorites first
